# Sistema de Gestión de Biblioteca Académica

## Descripción
Proyecto desarrollado en Java con Eclipse para la materia **Programación I (1S3212)**.  
Simula el funcionamiento de un sistema académico de préstamos de libros, con usuarios, ejemplares, penalizaciones y reportes.
